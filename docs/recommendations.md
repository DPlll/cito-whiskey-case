## Why
- Broad enough for a boutique shop
- Simple enough to manage at launch
- Useful for tasting flights

## Cluster Labels
- `Sherried & Sweet`: rich and smooth
- `Classic Smoky-Sweet`: balanced core shelf
- `Light & Approachable`: beginner-friendly
- `Herbal & Crisp`: exploratory styles
- `Maritime Peat`: smoky coastal styles

## Next Task
Build a simple tasting feedback system.

1. Collect a rating for each whiskey tasted.
2. Tag each customer by preferred style.
3. Track which tasting bottles turn into purchases.

## Other Uses of Analytics
- Tasting events: compare ratings and purchases by cluster.
- New supplier bottles: see if the bottle fills a gap or duplicates an existing style.
- Customer questions: use the cluster map to recommend similar bottles.
- Customer data: collect only basic preference and contact information.

## Limitation
The source data does not include cost, margin, or current distributor availability.
