# The Whiskey Case Slide Content

## Slide 1: Title
**The Whiskey Case**

- Kathryn Finnegan's boutique spirits shop near Harvard Square
- Goal: recommend an opening scotch assortment

## Slide 2: Data
- Used `scotch.xlsx` from `ScotchData.zip`
- Read the bundled readme files and assignment PDF
- Dataset used: 109 whiskies and 68 sensory variables

## Slide 3: Method
- Clustered whiskies by sensory descriptors
- Used a simple 5-cluster solution
- Kept the method easy to explain
