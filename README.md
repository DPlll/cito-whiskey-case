# cito-whiskey-case

Business analytics case project for The Whiskey Case.

## Run the analysis

```powershell
python src/whiskey_case_analysis.py
```

Or just double-click `run_me.bat`.

## Key outputs
- `run_me.bat`: one-click run
- `presentation/whiskey_case_slide_content.md`: slide-ready content
- `exports/figures/recommended_mix.png`: simplest chart
- `exports/tables/cluster_summary.csv`: basic cluster results
- `supporting_documents/`: source PDF, zip, workbook, and extracted support files
